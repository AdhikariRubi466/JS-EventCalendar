# JS Event Calendar 📅

A simple and interactive **JavaScript Event Calendar** web application built using **HTML, CSS, and JavaScript**.  
This project allows users to view calendar dates and manage events in an easy and user-friendly interface.

---

## 🚀 Features
- Monthly calendar view
- Previous and next month navigation
- Add events to specific dates
- Clean and responsive user interface
- Easy to use

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## ▶️ How to Run
1. Clone or download the repository
2. Open the project folder
3. Double-click `index.html` to open in browser

---

## 🎯 Purpose
This project is created for practicing JavaScript concepts such as:
- DOM manipulation
- Event handling
- Date and calendar logic
- UI design
